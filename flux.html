feel the Flux

Flux is a fast, open-source, Linux-based operating system designed with speed in mind. You will never miss a thing with its Android app compatibility. Security is the key as you have the choice of removing Google's services in favour of FOSS apps, too.
offline_bolt

Fast

FluxOS is built on the latest version of the Linux kernel with the best toolchain optimisations. Every device that runs Flux can reach its full potential.

 lock_open

Open

FluxOS was created by a community eager to escape from corporate dominion over tech. A large open-source app store is waiting for you.

 security

Secure

FluxOS uses its own encrypted file system called Flux. While slightly reducing transfer speeds, every file is automatically encrypted.

 apps

Apps

Flux uses a special shell to run android apps securely. You can use the same shell to run Virtual Machines with full virtualization.
